Related Software
  • MetaPost: A spin off from MetaFont, and the original inspiration for PyScript. You can do some really cool things with MetaPost, but the scripting language is awkward.
  • GLE: Graphics Layout Engine. Very similar in concept to Pyscript, but has it's own scripting language that is close to Postscript. It uses LaTeX-like expressions without actually using LaTeX. I got fustrated with it and decided to write my own ;)
  • PYX: A very similar project to pyscript. Some of the contructions are closer to postscript than the object-orientation of pyscript. Well worth checking out.
